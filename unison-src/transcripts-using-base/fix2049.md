Tests some capabilities for catching runtime exceptions.

```unison
catcher : '{IO} () ->{IO} Result
catcher act =
  handle tryEval act with cases
    { raise _ -> _ } -> Ok "caught"
    { _ } -> Fail "nothing to catch"

tests _ =
  [ catcher do
      match None with Some x -> x
  , catcher do
      1/0
      ()
  , catcher '(bug "testing")
  ]
```

.> add
.> io.test tests
